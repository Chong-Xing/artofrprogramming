## Question 01 (Multiple-Choice; p. 3)

A pdf file (a histogram of 100 values) was created using the following R code. Which line of the script causes the file to be writen to the disk?

A. > pdf("filename.pdf")
B. > hist(rnorm(100))
C. > dev.off()
D. all of the above

Answer: C

## Question 02 (Multiple-Choice; p. 4)

We created a vector q by using the following R code. What is the total number of elements in q?

> x <- c(1, 4, 8)
> q <- c(x, 5)

A. 3
B. 4
C. 5
D. 6

Answer: B
